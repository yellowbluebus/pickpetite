# PickPetite Landing Page

A beautiful, minimalist landing page for the PickPetite app — blending European design with Japanese minimalism.

## Quick Start

1. **Local Preview**
   ```bash
   cd pickpetite
   python3 -m http.server 8080
   ```
   Then open http://localhost:8080 in your browser.

2. **Deploy to GitHub Pages** (Free)
   - Create a new GitHub repository
   - Push this folder to the repository
   - Go to Settings → Pages → Select "main" branch
   - Your site will be live at `https://yourusername.github.io/pickpetite`

3. **Deploy to Netlify** (Free)
   - Drag and drop this folder to [netlify.com/drop](https://app.netlify.com/drop)
   - Or connect your GitHub repo for automatic deploys

4. **Connect Custom Domain**
   - Add your `pickpetite.com` domain in your hosting provider's settings
   - Update DNS records as instructed

## Amazon Associates Setup

Once deployed:
1. Go to [affiliate-program.amazon.com](https://affiliate-program.amazon.com)
2. Submit your deployed URL (e.g., `https://pickpetite.com`)
3. You'll get your Associate Tag immediately
4. Update affiliate links in your content
